# AI-профориентация (Mobile App + AI)

## Цель проекта
Создание мобильного приложения для профориентации студентов колледжей.
Студент сканирует QR-код, проходит интеллектуальное тестирование, после чего система
на основе ИИ подбирает профессии и колледжи, наиболее подходящие под его профиль.

Проект ориентирован на:
- осознанный выбор профессии
- снижение отсева студентов
- аналитику для учебных заведений

## Ключевые особенности
- адаптивное тестирование (ИИ подбирает вопросы)
- многомерный анализ (интересы, способности, личность, мотивация)
- отсутствие авторизации (быстрый вход)
- сбор данных для дальнейшего анализа
- масштабируемая архитектура

## Технологии
Frontend:
- Flutter **или** React

Backend:
- Node.js + Express

Database:
- Firebase (Firestore)

AI:
- aiTag-векторизация
- взвешенное сопоставление профессий
- LLM-интерпретация результатов (опционально)

## Как работает
1. Студент сканирует QR-код
2. Проходит тест (≈30 базовых вопросов)
3. ИИ анализирует ответы и формирует дополнительные уточняющие вопросы (адаптивное тестирование)
4. Вводит минимальные данные (ФИО, школа, класс)
5. Система формирует AI-профиль на основе всех ответов
6. ИИ подбирает профессии и объясняет выбор
